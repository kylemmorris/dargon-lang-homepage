Introduction

Dargon is a statically-typed, imperitive, extendable programming language with a focus on expressiveness and safety. It compiles to an intermediate language (IL) which runs on a stack-based virtual machine called the Dargon Virtual Machine (DVM). It takes inspiriation from a lot of modern programming language design, including some aspects of object-oriented, functional, and scripting languages.
